# 柏厚生活小程序一期核心规划（业务汇报版）

> **导语**：本方案旨在快速落地柏厚生活“卖空间、卖场景、卖全案、卖交付”的品牌理念，以轻量、高效的方式搭建线上选品与引流阵地。

## 1. 核心定位与业务价值

- **战略转化直达**：彻底打破传统“卖单品”的货架模式，让客户在线上就能直观感受到我们的空间设计能力与生活场景。
- **精准赋能双端**：
  - **对C端（业主）**：提供本地化的产品展示，所见即所得，快速建立信任并留下联系方式。
  - **对B端（设计师）**：提供专业的“素材弹药库”，支持下载高清元素图与场景图，直接用于酷家乐等软件出图，提高提案效率。

## 2. 一期核心功能清单

*注：一期我们将聚焦“展示、选品、引流”，暂缓复杂的后端交易与履约闭环，确保系统能最快速度上线并投入业务使用。*

1. **智能城市定位**
   - 用户进入小程序自动获取所在城市，展示当地可售商品与服务，避免无效咨询。
2. **多维“场景化”选品中心**
   - **按品类寻宝**：完整覆盖最新梳理的瓷砖、卫浴、定制、电器及辅材等 **34项核心品类**。
   - **按空间体验**：支持按 **7大核心空间**（玄关、客餐、卧室、书房、阳台、厨卫、衣帽间）沉浸式选品。
   - **按场景共鸣**：支持按 **7大主题场景**（亲子、收纳、极简、治愈、休闲、办公、养老）匹配产品方案。
3. **设计师专属“特权”通道**
   - 后台为设计师单开授权账号，普通用户不可见。
   - 设计师登录后，可查看专属折扣价，且产品详情页会多出**“下载素材”**按钮，一键获取高清原文件用于设计软件。
4. **极简意向留资**
   - 砍掉繁琐的注册流程，客户看到心仪产品或方案，一键提交“电话+意向”。
   - 预留轻量的量尺、安装等服务预约入口，线索直接流转给销售团队跟进。

## 3. 本周演示计划与待确认事项

- **本周目标**：在**本周五前**搭建出小程序一期的核心可点击框架。
- **演示方案**：先不上全量数据，而是选取1-2个核心品类（如瓷砖、洁具），走通“首页展示 -> 场景选品 -> 查看详情 -> 设计师下载素材”的完整流转。
- **待刘总确认**：届时请刘总体验该样板流程，重点判断**交互逻辑与业务展示方向**是否符合预期。确认无误后，下周将全面批量录入所有物料和类目数据。
