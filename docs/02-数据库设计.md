**核心实体关系**：
- Region ↔ Product：多对多（通过 `product.regions` JSON 数组实现，无中间表）
- Category → Product：一对多
- Product → Media：一对多
- WxUser → Lead / Appointment / Order：一对多
- Order → OrderItem：一对多
- Product → OrderItem：一对多
- DesignerGroup → Designer：一对多
- Designer → DownloadLog：一对多

### 2.1 bh_region（区域表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---|:---|
| `region_id` | varchar(32) | PK | 区域标识（如 `foshan`、`chengdu`） |
| `region_name` | varchar(64) | NOT NULL | 展示名称（如"佛山"） |
| `province` | varchar(32) | NOT NULL | 省份 |
| `center_lat` | decimal(10,6) | | 中心纬度 |
| `center_lng` | decimal(10,6) | | 中心经度 |
| `is_active` | tinyint(1) | DEFAULT 1 | 是否启用 |
| `sort_order` | int | DEFAULT 0 | 排序 |
| `create_by` | varchar(64) | | 创建者 |
| `create_time` | datetime | | 创建时间 |
| `update_by` | varchar(64) | | 更新者 |
| `update_time` | datetime | | 更新时间 |
| `remark` | varchar(500) | | 备注 |

```sql
CREATE TABLE bh_region (
    region_id     VARCHAR(32)    NOT NULL COMMENT '区域标识',
    region_name   VARCHAR(64)    NOT NULL COMMENT '展示名称',
    province      VARCHAR(32)    NOT NULL COMMENT '省份',
    center_lat    DECIMAL(10,6)  DEFAULT NULL COMMENT '中心纬度',
    center_lng    DECIMAL(10,6)  DEFAULT NULL COMMENT '中心经度',
    is_active     TINYINT(1)     NOT NULL DEFAULT 1 COMMENT '是否启用',
    sort_order    INT            NOT NULL DEFAULT 0 COMMENT '排序',
    create_by     VARCHAR(64)    DEFAULT '' COMMENT '创建者',
    create_time   DATETIME       DEFAULT NULL COMMENT '创建时间',
    update_by     VARCHAR(64)    DEFAULT '' COMMENT '更新者',
    update_time   DATETIME       DEFAULT NULL COMMENT '更新时间',
    remark        VARCHAR(500)   DEFAULT NULL COMMENT '备注',
    PRIMARY KEY (region_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='区域表';
```

### 2.3 bh_product（商品表）

| 字段 | 类型 | 约束 | 说明 |
|:---|:---|:---|:---|
| `id` | bigint | PK, AUTO_INCREMENT | 商品主键 |
| `name` | varchar(128) | NOT NULL | 商品名称 |
| `sku_code` | varchar(64) | UNIQUE, NOT NULL | 唯一 SKU 编码 |
| `brand` | varchar(64) | | 品牌 |
| `model` | varchar(64) | | 型号 |
| `category_id` | bigint | NOT NULL | 品类 ID |
| `guide_price` | decimal(10,2) | | 指导价（元） |
| `price_unit` | varchar(16) | | 计价单位（如"元/套"） |
| `designer_discount` | decimal(3,2) | DEFAULT 1.00 | 设计师折扣率（0.50~1.00） |
| `regions` | json | | 可售区域 JSON 数组，如 `["foshan","chengdu"]` 或 `["ALL"]` |
| `space_tags` | json | | 空间标签 JSON 数组 |
| `scene_tags` | json | | 场景标签 JSON 数组 |
| `spec_params` | json | | 规格参数 JSON 对象 |
| `description` | text | | 商品描述（富文本） |
| `status` | varchar(16) | DEFAULT 'draft' | 状态：draft/on_shelf/off_shelf/archived |
| `sort_order` | int | DEFAULT 0 | 排序权重 |
| `del_flag` | char(1) | DEFAULT '0' | 删除标志（0正常 1删除，若依风格） |
| `create_by` | varchar(64) | | 创建者 |
| `create_time` | datetime | | 创建时间 |
| `update_by` | varchar(64) | | 更新者 |
| `update_time` | datetime | | 更新时间 |

```sql
CREATE TABLE bh_product (
    id                  BIGINT         NOT NULL AUTO_INCREMENT COMMENT '商品主键',
    name                VARCHAR(128)   NOT NULL COMMENT '商品名称',
    sku_code            VARCHAR(64)    NOT NULL COMMENT 'SKU编码',
    brand               VARCHAR(64)    DEFAULT NULL COMMENT '品牌',
    model               VARCHAR(64)    DEFAULT NULL COMMENT '型号',
    category_id         BIGINT         NOT NULL COMMENT '品类ID',
    guide_price         DECIMAL(10,2)  DEFAULT NULL COMMENT '指导价(元)',
    price_unit          VARCHAR(16)    DEFAULT NULL COMMENT '计价单位',
    designer_discount   DECIMAL(3,2)   NOT NULL DEFAULT 1.00 COMMENT '设计师折扣率',
    regions             JSON           DEFAULT NULL COMMENT '可售区域JSON数组',
    space_tags          JSON           DEFAULT NULL COMMENT '空间标签JSON数组',
    scene_tags          JSON           DEFAULT NULL COMMENT '场景标签JSON数组',
    spec_params         JSON           DEFAULT NULL COMMENT '规格参数JSON',
    description         TEXT           DEFAULT NULL COMMENT '商品描述',
    status              VARCHAR(16)    NOT NULL DEFAULT 'draft' COMMENT '状态(draft/on_shelf/off_shelf/archived)',
    sort_order          INT            NOT NULL DEFAULT 0 COMMENT '排序权重',
    del_flag            CHAR(1)        NOT NULL DEFAULT '0' COMMENT '删除标志(0正常 1删除)',
    create_by           VARCHAR(64)    DEFAULT '' COMMENT '创建者',
    create_time         DATETIME       DEFAULT NULL COMMENT '创建时间',
    update_by           VARCHAR(64)    DEFAULT '' COMMENT '更新者',
    update_time         DATETIME       DEFAULT NULL COMMENT '更新时间',
    PRIMARY KEY (id),
    UNIQUE KEY uk_sku_code (sku_code),
    KEY idx_category_status (category_id, status),
    KEY idx_status_sort (status, sort_order, create_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='商品表';

-- JSON 字段 GIN 等效索引（MySQL 8.0 多值索引）
-- 注意：MySQL 8.0.17+ 支持多值索引
ALTER TABLE bh_product ADD INDEX idx_regions ((CAST(regions AS CHAR(32) ARRAY)));
ALTER TABLE bh_product ADD INDEX idx_space_tags ((CAST(space_tags AS CHAR(32) ARRAY)));
ALTER TABLE bh_product ADD INDEX idx_scene_tags ((CAST(scene_tags AS CHAR(32) ARRAY)));
```

## 4. 数据安全

| 安全措施 | 说明 |
|:---|:---|
| 手机号加密 | 所有手机号字段使用 AES-256 加密存储，同时存储 SHA-256 哈希值用于查询匹配 |
| 软删除 | 商品表使用 `del_flag` 标记删除（若依风格），业务表使用 status 终态 |
| 数据备份 | MySQL 开启 binlog，每日全量备份 + 增量备份 |
| 敏感字段脱敏 | 后台展示手机号时中间四位用 `*` 替代 |
